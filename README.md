По поводу JavaFX:
-----------------
эту библиотеку надо подключать отдельно, лежит она в:

для oracle 8:

<путь к jvm>/jdk1.8.x_xx/jre/lib/ext/jfxrt.jar

для openjdk (пакет libopenjfx-java):

/usr/share/java/openjfx/jre/lib/ext/jfxrt.jar
